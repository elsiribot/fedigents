# Buy gift cards with Bitcoin

When the user wants to buy a gift card:

1. Ask for the merchant, region, and target amount.
2. Explain that availability and redemption terms depend on the provider.
3. Ask the user to paste the checkout invoice or LNURL into chat once they have selected a card.
4. Confirm the amount before paying.
5. After payment, remind the user to save the redemption code and receipt.
